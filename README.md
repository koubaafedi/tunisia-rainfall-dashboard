# Tunisian Groundwater Monitoring Dashboard

This dashboard visualizes the estimated groundwater levels in Tunisia based on daily rainfall data.

## Setup

1.  **Install Python:** Ensure you have Python installed on your system.
2.  **Install Dependencies:** Open your terminal in this folder and run:
    ```bash
    pip install -r requirements.txt
    ```

## Running the App

1.  Run the Streamlit app:
    ```bash
    streamlit run app.py
    ```
2.  The app will open automatically in your web browser.

## Data Sources

- **Rainfall Data:** Daily updates from [AgriData.tn](https://catalog.agridata.tn).
- **Coordinates:** Station list from AgriData.tn.
